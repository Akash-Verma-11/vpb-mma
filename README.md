# VPB-MMA — Books & Stationery E-Commerce Platform

A cloud-native microservices e-commerce system on AWS EKS, built with Terraform,
Kubernetes, Jenkins CI/CD, and Prometheus/Grafana monitoring.

## Services
- web-frontend
- catalog-service
- customer-service
- order-service
